## MANIPULATE A SHAPE
This is a simple webpage with a simple box with a simple objective: manipulate the box. The following commands are available:
1. size:[px]
2. position:[x,y]
3. color:[color]
4. height:[p]
5. length:[px]
6. width:[px]
7. reset:[all|color|position|size]
